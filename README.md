# Enmity

A Discord library for Elixir.

## Installation

The package can be installed by adding `enmity` to your list of dependencies in `mix.exs`:

```elixir
def deps do
  [
    {:enmity, "~> 0.1.0"}
  ]
end
```

Documentation can be found at [https://hexdocs.pm/enmity](https://hexdocs.pm/enmity).
